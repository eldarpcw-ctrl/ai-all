# AI All — Руководство по искусственному интеллекту

🌐 **Live Demo:** [https://eldarpcw-ctrl.github.io/ai-all](https://eldarpcw-ctrl.github.io/ai-all)

> Полное практическое руководство по применению ИИ в 12 отраслях — с готовыми промптами, топовыми моделями и рабочими сценариями.

---

## 🚀 Возможности

- **12 отраслей** — Медицина, Финансы, Образование, Маркетинг, Право, Программирование, Творчество, HR, Производство, Торговля, Кибербезопасность, Наука
- **18+ моделей ИИ** с фильтрацией по типу (Текст, Код, Изображения, Аудио, Видео)
- **150+ практик** — готовые промпты и пошаговые сценарии
- **PWA** — устанавливается на телефон и компьютер, работает офлайн
- Премиальный тёмный дизайн, анимации, glassmorphism

## 📦 Структура

```
ai-all/
├── index.html          # Главная страница
├── style.css           # Стили (dark theme)
├── app.js              # Логика и данные
├── manifest.json       # PWA манифест
├── sw.js               # Service Worker
└── icons/              # Иконки PWA
```

## 🛠 Локальный запуск

```bash
# Python
python -m http.server 3000

# Затем откройте: http://localhost:3000
```

## 📱 Установка как PWA

Откройте в Chrome → нажмите иконку установки в адресной строке.
